Detailed Description

template<size_t N>
class kaori::DualBarcodes< N >

Handler for dual barcodes.

One of the paired reads contains a barcode from one pool of options, while the other read contains a barcode from another pool. Unlike CombinatorialBarcodesPairedEnd, the combinations are known in advance, typically corresponding to specific pairs of genes. This handler will capture the frequencies of each barcode combination.

Template Parameters
NSize of the bitset to use for each constant template. The maximum size of the template is defined as N / 4, see ConstantTemplate for details.
