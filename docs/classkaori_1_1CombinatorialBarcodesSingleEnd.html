Detailed Description

template<size_t N, size_t V>
class kaori::CombinatorialBarcodesSingleEnd< N, V >

Handler for single-end combinatorial barcodes.

Each read contains a construct with multiple variable regions, where each region contains a barcode from a different pool of options. This handler will capture the frequencies of each barcode combination.

Template Parameters
NSize of the bitset to use for each constant template. The maximum size of the template is defined as N / 4, see ConstantTemplate for details.
VNumber of variable regions in the construct.
