Detailed Description

Library of known sequences for variable regions.

This supports exact and mismatch-aware searches for known sequences. Mismatches may be distributed anywhere along the length of the sequence. Instances of this class use caching to avoid searching the MismatchTrie when a mismatching sequence has been previously encountered.
