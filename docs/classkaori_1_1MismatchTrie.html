Detailed Description

Base class for the trie structure.

Given a (typically read-derived) sequence, the trie allows us search for the best match to known sequences in a SequenceSet. Any number of mismatches are supported, though the actual search itself depends on how the mismatches can be distributed throughout the length of the sequence.
