Detailed Description

template<size_t N>
class kaori::CombinatorialBarcodesPairedEnd< N >

Handler for paired-end combinatorial barcodes.

One of the paired reads contains a barcode from one pool of options, while the other read contains a barcode from another pool. Typically, these combinations are assembled randomly by library construction. This handler will capture the frequencies of each barcode combination.

Template Parameters
NSize of the bitset to use for each constant template. The maximum size of the template is defined as N / 4, see ConstantTemplate for details.
